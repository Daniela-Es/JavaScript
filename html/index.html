Una variable puede definirse facilmente como un espacio de memoria donde se almacena un dato

Puede pensarse en una variable como una caja donde podemos guardar un dato, con el nombre del mismo para poder utilizarlo despues

El uso de variables permite que nuestro programa sea mas general. Podemos declarar variables para utilizarlas a lo largo del programa dandoles valores definidos que pueden cambiar dependiendo de nuestras necesidades

  1. Utilizamos la palabra reservada var.
  2. Damos nombre a la variable
  3. Signo de igual para definirla
  4. Damos un valor inicial a la variable

Ejemplo:

var nombre = "Daniela"

Repaso Consola del Navegador

Es una herramienta para desarrolladores, usuarios avanzados o cualquier usuario que desee conocer mas al fondo el funcionamiento de una pagina web. En ella se muestran mensajes de informacion, error o alerta que se reciben al cargar la informacion para la pagina.

Se utiliza para depurar las paginas, ya que nos muestra variables mientras se ejecutan para ver si estamos obteniendo el resultado esperado. Tambien nos permite inspeccionar los elementos de la pagina de manera individual o revisar la responsividad de la misma.

Para obtenerla simplemente debes dar click derecho sobre cualquier parte de la pagina web y encontraras la opcion "Inspect" o "Inspeccionar". Alli mismo encontraras la combinacion de teclas que abren tu consola, la cual cambia dependiendo del navegador que uses y el sistema operativo con el que trabajas.
